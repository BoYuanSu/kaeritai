# 快回家 (Kaeritai) - 圖片合成工具

一個簡單直觀的圖片合成工具，靈感來自 [@fukuroupiyopiyo](https://x.com/fukuroupiyopiyo) 的推文。

**線上 Demo:** https://boyuansu.github.io/kaeritai/

## 功能特性

- 📸 圖片合成 - 輕鬆合成風景與寵物照片
- 🎨 覆蓋層操作 - 自由調整覆蓋層位置和大小
- 🔄 覆蓋層縮放 - 靈活的縮放支持
- 🎭 覆蓋層遮罩 - 創建圓形、方形的遮罩效果
- 🌈 漸變遮罩 - 支持漸變遮罩效果

## 技術棧

- **前端框架**: React 19 + TypeScript
- **構建工具**: Vite
- **代碼品質**: ESLint
- **包管理**: pnpm

## 開發設置

### 安裝依賴
```bash
pnpm install
```

### 本地開發
```bash
pnpm dev
```

### 構建生產版本
```bash
pnpm build
```
